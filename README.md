# CS-255-System-Analysis-and-Design

**Overview**

This repository contains the system analysis and design artifacts created for CS-255: System Analysis and Design. The project demonstrates the process of gathering requirements, modeling system behavior, and designing a software architecture using UML diagrams.

**Project Components**

The project includes several components used to model and design a software system:

- System design documentation  
- Use case diagrams  
- Class diagrams  
- Sequence diagrams  
- System architecture diagrams  

These artifacts demonstrate the planning phase of software development and illustrate how software systems can be designed before implementation.

**Skills Demonstrated**

This project demonstrates several important software engineering skills including:

- Requirements analysis  
- UML modeling  
- System architecture planning  
- Documentation and technical communication
